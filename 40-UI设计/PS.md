# 快捷键

`ctrl` +滚轮：左右移动

`shift` +滚轮：上下移动

`alt` +滚轮：放大缩小

`ctrl` + `n` ：新建文档

`alt` + `ctrl` + `i` ：更改图像大小

`ctrl` + `s` ：存储

`ctrl` + `k` ：首选项

`F7` ：图层面板

`ctrl` + `J` ：等位复制

`ctrl` + `T` ：自由变换

`shift` + `alt` ：以中心点对称且等比缩放

`ctrl` + `alt` + `F` ：重复上一步滤镜

`ctrl` + `D` ：取消选区
